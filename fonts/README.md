# Fonts

Drop the **Retroica** display font here so the brand wordmark and big numbers use it.

1. Download from https://www.dafont.com/retroica.font (free for personal use)
2. Save as `Retroica.ttf` (and/or `Retroica.woff2`) in this folder.

Without it, the dashboard falls back to **Space Grotesk** automatically.
